# my-neovim-config
My neovim config files

[How it looks](preview.png)

Configs for each plugin are stoerd seperately in the plugin_conf directory, which are sourced by init.vim at the end.

This is the default [Gruvbox Theme](https://github.com/morhetz/gruvbox), I have only changed the background to match my terminal color scheme.(Lavandula from [Gogh](https://github.com/Gogh-Co/Gogh))

If you're using gnome, you can get the blurred windows patch of mutter from [mutter-rounded](https://github.com/yilozt/mutter-rounded)
